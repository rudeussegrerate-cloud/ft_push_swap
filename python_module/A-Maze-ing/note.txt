2) #Makefile

- mandatory lint

a) install : installe les dépendances du projet à l’aide de pip, uv, pipx ou de tout autre gestionnaire de paquets
de votre choix.

b) • run : exécutez le script principal de votre projet (par exemple, via l'interpréteur Python).

c) • debug : exécutez le script principal en mode débogage à l'aide du débogueur intégré à Python (par exemple,
pdb).

d) clean : supprimez les fichiers temporaires ou les caches (par exemple, __pycache__, .mypy_cache) afin de
maintenir l'environnement du projet propre.

e) • lint : Exécutez les commandes flake8 et mypy avec les options --warn-return-any
--warn-unused-ignores --ignore-missing-imports --disallow-untyped-defs
--check-untyped-defs
f) • lint-strict (facultatif) : Exécutez les commandes flake8 et mypy avec l'option --strict

- gestion des erreur


5) Configuration file format:

key = value :
Key 		Description 			Example
WIDTH 		Maze width (number of cells) 	WIDTH=20
HEIGHT 		Maze height 			HEIGHT=15
ENTRY 		Entry coordinates (x,y) 	ENTRY=0,0
EXIT 		Exit coordinates (x,y) 		EXIT=19,14
OUTPUT_FILE 	Output filename 		OUTPUT_FILE=maze.txt
PERFECT 	Is the maze perfect? 		PERFECT=True

- Vous pouvez ajouter des clés supplémentaires (par exemple, graine, algorithme, mode d'affichage) si cela s'avère utile.
Un fichier de configuration par défaut doit être disponible dans votre dépôt Git

Maze Requirements
- randomly generated
- 0 and 4 walls
- walls at the external borders.
- data must be coherent
- • Si le drapeau PERFECT est activé, le labyrinthe doit contenir exactement un chemin entre 
    l'entrée et la sortie (c'est-à-dire qu'il doit s'agir d'un labyrinthe parfait).
